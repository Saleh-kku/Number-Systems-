Introduction to Number Systems

Decimal (Base 10)

This is the most commonly used number system, which uses the digits 0 to 9.

Binary (Base 2)

This number system uses only two digits: 0 and 1. It is the fundamental number system used in digital electronics and computer systems.

Octal (Base 8)

This number system uses the digits 0 to 7. It is sometimes used as an intermediate step in converting between binary and hexadecimal.

Hexadecimal (Base 16)

This number system uses the digits 0 to 9 and the letters A to F to represent values from 0 to 15. It is commonly used in computer programming and digital electronics.

Converting from Binary, Octal, or Hexadecimal to Decimal

Binary to Decimal

Each bit in the binary number represents a power of 2 (20, 21, 22, etc.).

To convert a binary number to decimal, multiply each bit by its corresponding power of 2 and then add the results.

Example: Convert 1010.01 (binary) to decimal:

(1 × 23) + (0 × 22) + (1 × 21) + (0 × 20) + (0 × 2-1) + (1 × 2-2) = 8 + 0 + 2 + 0 + 0 + 0.25 = 10.25 (decimal)

Octal to Decimal

Each digit in the octal number represents a power of 8 (80, 81, 82, etc.).

To convert an octal number to decimal, multiply each digit by its corresponding power of 8 and then add the results.

Example: Convert 456.7 (octal) to decimal:

(4 × 82) + (5 × 81) + (6 × 80) + (7 × 8-1) = 256 + 40 + 6 + 0.875 = 302.875 (decimal)

Hexadecimal to Decimal

Each digit in the hexadecimal number represents a power of 16 (160, 161, 162, etc.).

To convert a hexadecimal number to decimal, multiply each digit by its corresponding power of 16 and then add the results.

Example: Convert A7F.C (hexadecimal) to decimal:

(10 × 162) + (7 × 161) + (15 × 160) + (12 × 16-1) = 2560 + 112 + 15 + 0.75 = 2687.75 (decimal)
